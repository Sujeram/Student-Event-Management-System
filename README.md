# Student Event Management System

## About

The Student Event Management System is a full-stack web application developed to simplify the management of college events. It allows administrators to create and manage events, while students can view available events and register through an interactive web interface.

## Features

- Student registration and login
- Create, update and delete events
- Event registration
- View upcoming events
- Dashboard for event management
- Responsive user interface
- MongoDB database integration

## Technologies Used

- Java
- Spring Boot
- React.js
- TypeScript
- Tailwind CSS
- HTML
- CSS
- MongoDB
- Maven
- REST API

## How to Run

1. Install Java 17 or later.
2. Install Node.js.
3. Install MongoDB.
4. Start the backend application.
5. Open the frontend folder and run:

```
npm install
npm run dev
```

6. Open the application in your browser.
